# Network Management & Troubleshooting Commands (Linux)

- **`ip addr`** – Display IP addresses and network interface details
- **`ip link`** – Show or change network interface status (up/down)
- **`ifconfig`** – Display or configure network interfaces (legacy)
- **`ip route`** – Show or modify the routing table
- **`route`** – Display or modify routing information (legacy)
- **`ping`** – Test network connectivity between hosts
- **`ss`** – Display active sockets and listening ports
- **`netstat`** – Show network connections, routing tables, and ports (legacy)
- **`traceroute`** – Trace the path packets take to a destination
- **`nmap`** – Scan hosts and services on a network
- **`arp`** – View or modify the ARP cache
- **`arp -a`** – Display IP-to-MAC address mappings
- **`nmcli`** – Manage NetworkManager connections from the command line
- **`iwconfig`** – Configure wireless network interfaces (legacy)
- **`hostname`** – Display or set the system hostname
- **`dig`** – Query DNS servers for name resolution
- **`nslookup`** – Test DNS name resolution
